Kotlin in Numbers

Shout out the answer!

How old is Kotlin?

Answer: Unveiled in 2011, version 1.0 released in 2016!

How many keywords does the language have?

Answer: 81 keywords across 4 categories
(Hard keywords, Soft keywords, Modifier keywords, and Special identifiers)

How many visibility modifiers does Kotlin have?

Answer: 4
(public, private, protected, and internal)

How many standard helper methods does the compiler generate for a data class?

(Excluding component functions like component1())

Answer: 3
(equals(), hashCode(), and toString())

What is the maximum number of abstract methods a functional interface can declare?

Answer: 1
(It must have exactly one abstract method, though it can have multiple non-abstract/default methods)
